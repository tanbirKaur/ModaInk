Returns & Exchange

Exchange will be done if the Items are returned / exchanged within 3 days of delivery.

The tags & labels should be intact and the original packaging is retained.

The item should be unused, unworn, undamaged & unwashed.

Returns will only be done in case of damaged / torn merchandise or if the packaging is tampered.

Exchange will only be done with a different size of the same design.

Custom Orders / Made to Order Products are not applicable for Return or Exchange.

No Cancellations of Orders will be accepted.
